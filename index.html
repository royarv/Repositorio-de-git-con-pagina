El recetario del arquitecto (Servicios AWS)

Rodrigo Arvizu Medina IRD 5A

¿Qué problema resuelven, cuándo usarlos y cuándo no?

EC2 RDS S3 Load Balancer Auto Scaling CloudFront DynamoDB Lambda

EC2 (Elastic Compute Cloud)

¿Qué problema resuelve?Permite ejecutar servidores virtuales sin hardware físico.
¿Cuándo lo usarías?Aplicaciones web, APIs, control total del sistema.
¿Cuándo NO lo usarías?Tareas pequeñas o procesos eventuales.
Ejemplo real de usoServidor web para una tienda en línea.
CostoMedio
Si desapareciera...Servidores físicos o VMs en otros proveedores.

RDS (Relational Database Service)

¿Qué problema resuelve?Administración automática de bases de datos relacionales.
¿Cuándo lo usarías?Datos estructurados y relaciones complejas.
¿Cuándo NO lo usarías?Datos no estructurados o proyectos pequeños.
Ejemplo real de usoBase de datos de usuarios y pedidos.
CostoCaro
Si desapareciera...Base de datos instalada manualmente en EC2.

S3 (Simple Storage Service)

¿Qué problema resuelve?Almacenamiento de archivos seguro y escalable.
¿Cuándo lo usarías?Imágenes, videos, backups.
¿Cuándo NO lo usarías?Ejecutar aplicaciones o bases de datos.
Ejemplo real de usoFotos de perfil de usuarios.
CostoBarato
Si desapareciera...Discos locales o servidores de archivos.

Load Balancer

¿Qué problema resuelve?Distribuye tráfico para evitar caídas.
¿Cuándo lo usarías?Múltiples servidores y alta disponibilidad.
¿Cuándo NO lo usarías?Un solo servidor.
Ejemplo real de usoSitio web con muchos usuarios simultáneos.
CostoMedio
Si desapareciera...Nginx o HAProxy manual.

Auto Scaling

¿Qué problema resuelve?Escala servidores automáticamente según demanda.
¿Cuándo lo usarías?Tráfico variable o picos inesperados.
¿Cuándo NO lo usarías?Tráfico estable.
Ejemplo real de usoVenta de boletos en eventos masivos.
CostoMedio
Si desapareciera...Escalado manual.

CloudFront

¿Qué problema resuelve?Entrega rápida de contenido global.
¿Cuándo lo usarías?Usuarios en distintas regiones.
¿Cuándo NO lo usarías?Apps internas.
Ejemplo real de usoStreaming y carga de imágenes.
CostoBarato
Si desapareciera...Otros CDNs o servidores regionales.

DynamoDB

¿Qué problema resuelve?Base de datos NoSQL escalable.
¿Cuándo lo usarías?Millones de operaciones rápidas.
¿Cuándo NO lo usarías?Consultas SQL complejas.
Ejemplo real de usoSesiones de usuarios.
CostoMedio
Si desapareciera...MongoDB o Cassandra.

Lambda

¿Qué problema resuelve?Ejecuta código sin administrar servidores.
¿Cuándo lo usarías?Eventos, APIs pequeñas.
¿Cuándo NO lo usarías?Procesos largos.
Ejemplo real de usoEnviar correos automáticos.
CostoBarato
Si desapareciera...Cron jobs en servidores.
